Your calculator is hidden from the world right now. Take your time to build it. Once you finish your work, mark the calculator as finished in the configuration. That will publish it to the list of calculators and it will be possible to embed it.

Use Joomla plugin, WordPress plugin or insert one of following codes into your HTML web page. It will embed this calculator, which has ID = {{calc.id}}.

<link rel="stylesheet" href="https://embed.calculoid.com/styles/main.css" />
<script src="https://embed.calculoid.com/scripts/combined.min.js"></script>
<div ng-app="calculoid" ng-controller="CalculoidMainCtrl" ng-init="init({calcId:{{calc.id}},apiKey:'{{user.apiKey}}'{{embedShowTitleText}}{{embedShowDescriptionText}}})" ng-include="load()"></div>
